# JB Auto — Navigation responsive avec Flexbox

Projet réalisé dans le cadre du module **HTML, CSS & Git** (Akieni Academy).  
**Auteur : HIRWA Jean Baptiste**

## Description

Site vitrine pour une concession automobile. Le focus du projet est une barre de navigation responsive qui se transforme en menu burger sur mobile, construite uniquement en **HTML** et **CSS**.

## Fonctionnalités

- Logo à gauche, menu centré, bouton **Contactez-nous** à droite
- Menu burger fonctionnel au clic (sans JavaScript)
- Panneau latéral avec overlay sombre sur mobile
- Section **Nos Véhicules** avec 3 cards
- Prix affichés en **FCFA**
- Dropdown sur le lien Véhicules (bonus)
- Animations fluides avec transitions CSS (bonus)

## Structure du projet

```
.

```

## Lancer le projet

1. Cloner le dépôt :
   ```bash
   git clone https://github.com/JBixx/JB-Voiture-auto.git
   ```
2. Ouvrir `index.html` dans un navigateur.

Aucune installation nécessaire.
